THE VOLUMETRIC JLSM ENGINE: MASTER BLUEPRINT

I. Repository & Governance Matrix (The Execution Shell)

The repository is structured to enforce a strict boundary between rigorous headless computation, visual playback, and autonomous agent coordination. To protect compute budgets, the CI/CD pipeline heavily prioritizes localized, self-hosted execution.

1. Directory Structure:

core_pbd/: The Rust-based headless physics engine. Contains the Position Based Dynamics (PBD) solver, the 1Hz cellular automata grids, and FFI (Foreign Function Interface) bindings via PyO3/Maturin.

fishtank_ui/: The Python/WebGL visualizer. Strictly a read-only consumer of headless telemetry.

.github/: The automated governance layer. Contains Issue/PR YAML templates, Dependabot configs, and selectively-gated workflows.

agent_logic/: Scripts for parsing Abstract Syntax Trees (ASTs) to feed token-efficient context to the Coding Squad.

docs/: The Ontological Commons, automatically synced to the GitHub Wiki.

2. CI/CD & Testing (The Lean Pipeline):

Path Gating: Cloud CI only triggers when corresponding domain files are altered (e.g., UI changes ignore the Rust compiler).

Pre-Commit Hooks: Local machines run standard fast-tests and linters. Cloud runners are bypassed for routine checks.

Self-Hosted QA (WSL): Heavy validations—specifically the 60Hz Criterion benchmarking and the "Dead World" 5,000-tick entropy simulations—execute on local hardware, burning zero cloud compute minutes.

AST Token Diet: Autonomous agents receive only function signatures and type definitions via AST stripping when planning features, eliminating token exhaustion.

II. The World Matrix (Environment & Thermodynamics)

The environment consists of a continuous vector space overlaid with discrete scalar grids and topological fields. The physics step runs at 60Hz; thermodynamic and grid systems run at 1Hz.

1. Topology (Signed Distance Field - SDF):

The ground is an SDF, allowing $O(1)$ collision checks and cheap raymarching.

Dynamic Erosion & Marine Snow: Detritus from dead organic matter sinks, accumulating as voxel density on the seabed. Deep currents erode the topology over time.

2. The 1Hz Thermodynamic Grids:

Illumination ($I$): A 3D voxel grid. Driven by a diurnal sine-wave at the surface ($Z=0$). Attenuates downward via Beer-Lambert law. Blocked by SDF terrain. Additively spiked by localized bioluminescence.

Temperature ($T$): Surface water is warm; deep water is freezing. When fluid currents intersect SDF terrain, they create Upwellings, dragging freezing, nutrient-dense water to the surface.

Oxygen ($O_2$): Generated by photosynthetic flora near the surface. Depleted by fauna muscle actuation. Causes localized asphyxiation if the population over-clusters.

Fluid Currents: A 3D Perlin vector field providing passive momentum.

III. Somatic Morphology (The JLSM Graph)

Entities are soft/rigid-body hybrids constructed via a Joint-Link-Skin-Muscle directed graph, solved via Rust's PBD engine.

1. Structural Components:

Joints: Volumetric nodes. Define rotational limits (hinge, ball-in-socket).

Links: Distance constraints. Governed by a compliance float ($\alpha$). $\alpha=0$ is rigid bone; $\alpha>0$ is elastic tissue. Every Link has structural_integrity. If integrity hits 0 (via damage or over-exertion), the Link snaps, physically severing the limb.

The Core Node: A singular, genetically designated Joint that anchors the neural matrix. If the Core Node takes direct fatal damage, systemic death is instantaneous.

2. The Skin Matrix (Boundary Layer):

Triangulated meshes spanning the Links, defining collision bounds and drag coefficients.

Insulation: A genetic thickness variable. High thickness negates thermal metabolic drain in deep water but geometrically increases entity mass and locomotion energy cost.

Specialized Nodes: Skins host sensory arrays, Bioluminescent emitters, Maws (for ingestion), and Gametangia (for contact copulation).

IV. The Cognitive Matrix (The Epigenetic Brain)

The entity's brain is a spatially routed, decentralized neural network with dynamic chronologies.

1. Bipartite Genetics & Spatial Routing:

Block A (Somatic): Encodes the physical JLSM graph.

Block B (Cognitive): Encodes the baseline neural weights ($W_{base}$) and 3D spatial coordinates of hidden neurons.

Inputs (sensors) and Outputs (muscles) map to the nearest physical hidden neuron in 3D space, preventing catastrophic I/O misalignment when an entity mutates a new limb.

2. Plasticity & Lamarckian Transfer:

Entities utilize Hebbian learning to shift weights during their lifetime, creating $W_{life}$.

Upon reproduction, the offspring inherits an Epigenetic Imprint: a mathematically weighted blend of the parent's $W_{base}$ and $W_{life}$, forming "genetic instinct."

3. Elastic Perception (The Flicker Fusion Rate):

Neural evaluations are decoupled from the 60Hz physics tick via an Accumulator Latch.

Bounds: $\omega_{max} = 60Hz$, $\omega_{min} = 0.2Hz$.

The Arousal Latch: An internal neural output modulating clock speed dynamically. Detecting a shadow or triggering a Nociceptor spikes the latch, ramping perception toward 60Hz.

Metabolic Debt: The caloric cost of running the neural matrix scales exponentially as it approaches 60Hz.

V. Ecological & Evolutionary Pressures

The simulation strictly enforces absolute thermodynamic conservation. There is no programmed fitness function. Population is capped at $N=100$.

1. Metabolism, Damage & Ingestion:

A specialized Skin node (Maw) colliding with prey applies massive negative impulse to the target's structural_integrity.

Thermodynamic Ingestion: $E_{gained} = Mass_{consumed} * \text{Entropy\_Coefficient}$ (e.g., 0.8).

Excess internal energy can regenerate structural_integrity of damaged Links, but only if the entity enters a low-perception resting state.

2. Sensory Inputs:

Directional Eye: Casts an SDF raymarch along the Skin's normal vector. Senses shadows via bounding-box cylinder casts against other entities. Photons are buffered between neural ticks.

Nociceptors: Detect localized drops in structural_integrity.

Gyroscopes: Dot product against the universal Z-axis.

3. Lifecycle (Senescence & Reproduction):

Aging: cellular_degradation_rate causes baseline metabolic costs to creep upward over time.

Isogamous Reproduction: Genderless. Achieved via sustained contact of Gametangia nodes, or by expending massive energy to release broadcast gametes into the Perlin current field.

Genetic Drift: Constant background radiation applies micro bit-flipping to genomes upon crossover.
